一. 宏

  1. do{}while(0) 宏的用处?

    避免调用者出现使用错误，保证程序执行的一致性。
    例如：

    #define FOO() foo1();foo2();

    调用代码：

    if (a > 0)
    FOO();

    会被扩展成:

    if (a > 0)
    foo1();foo2();

    这时就产生了一个问题，若是a <= 0，foo2()仍然会被调用。
    正确的写法：

    #define FOO() \
        do{ \
            foo1(); \
            foo2(); \
        }while(0); \
